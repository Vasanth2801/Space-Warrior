ABOUT THIS PACK

This pack contains a refreshing collection of clean 16×16 pixel-art health UI elements, including:

• Bar styles: Horizontal, Vertical, Block-style
• Animation variants: Filling, draining, removal effects
• Health icons: Full, half, low
• Fill states: Full, partial (multiple levels), empty
• All sprites: 16×16, pixel-perfect, and ready for retro-style projects

These sprites are ideal for:

• 2D survival games
• Health mechanics
• UI / UX bars and icons
